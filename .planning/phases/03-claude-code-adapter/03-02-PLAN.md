    1. Create `src-tauri/src/commands/claude.rs` with three Tauri commands:

    **spawn_claude_task** — `#[tauri::command] #[specta::specta]`
    - Parameters: `prompt: String, project_dir: String, on_event: Channel<OutputEvent>, state: tauri::State<'_, AppState>`
    - Returns: `Result<String, String>` (task_id)
    - Implementation:
      a. Retrieve API key via `tokio::task::spawn_blocking(|| credentials::keychain::get_api_key()).await`
      b. Build command with `adapters::claude::build_command(&prompt, &project_dir, &api_key)`
      c. Call `process::manager::spawn_with_env(cmd, args, cwd, env_vars, on_event, state).await`
      d. Return task_id

    **set_claude_api_key** — `#[tauri::command] #[specta::specta]`
    - Parameters: `key: String`
    - Returns: `Result<(), String>`
    - Implementation: `tokio::task::spawn_blocking(move || credentials::keychain::set_api_key(&key)).await`
    - IMPORTANT: Validate key format starts with "sk-ant-" before storing. Return error if invalid format.

    **has_claude_api_key** — `#[tauri::command] #[specta::specta]`
    - Parameters: none
    - Returns: `Result<bool, String>`
    - Implementation: `tokio::task::spawn_blocking(|| credentials::keychain::has_api_key()).await`

    2. Update `src-tauri/src/commands/mod.rs`:
