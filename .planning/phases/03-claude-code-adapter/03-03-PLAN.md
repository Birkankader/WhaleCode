    - Test: parseClaudeEvent returns null for non-JSON string
    - Test: parseClaudeEvent returns null for empty string
    - Test: formatClaudeEvent formats message event as readable text
    - Test: formatClaudeEvent formats tool_use event as "[Tool: name] input"
    - Test: formatClaudeEvent formats result event with status and cost
    - Test: formatClaudeEvent returns raw text for unparseable lines
  </behavior>
  <action>
    1. Create `src/lib/claude.ts` with:

    **Types** (mirror Rust ClaudeStreamEvent but in TypeScript):
    ```
    interface ClaudeStreamEvent {
      type: 'init' | 'message' | 'tool_use' | 'tool_result' | 'result' | 'stream_event';
      session_id?: string;
      role?: string;
      content?: Array<{ type: string; text?: string; name?: string; input?: unknown }>;
      name?: string;
      input?: unknown;
      output?: string;
      status?: string;
      result?: string;
      is_error?: boolean;
      duration_ms?: number;
      num_turns?: number;
      total_cost_usd?: number;
    }
    ```

    **parseClaudeEvent(line: string): ClaudeStreamEvent | null** — JSON.parse in try/catch, returns null on failure.

    **formatClaudeEvent(line: string): string** — Parses line, formats based on event type:
    - `init`: "[Session started]"
    - `message`: Extract text from content blocks, join with newlines
    - `tool_use`: "[Tool: {name}] {JSON.stringify(input)}"
    - `tool_result`: "[Result] {output}"
    - `result`: "[Done] status={status}, turns={num_turns}, cost=${total_cost_usd}" (or error if is_error)
    - unparseable: return raw line as-is

    2. Create `src/hooks/useClaudeTask.ts`:
    - Follow the pattern from `useProcess.ts`
    - Export `useClaudeTask()` hook returning `{ spawnTask, isRunning, hasApiKey, checkApiKey }`
    - `spawnTask(prompt: string, projectDir: string)` — calls `commands.spawnClaudeTask` from bindings
    - `checkApiKey()` — calls `commands.hasClaudeApiKey`, updates `hasApiKey` state
    - The hook receives an `onEvent` callback prop for handling OutputEvents
    - Inside the hook, the stdout handler should call `formatClaudeEvent` to convert NDJSON to readable text before passing to the terminal

    3. Create `src/tests/claude.test.ts` with Vitest tests for parseClaudeEvent and formatClaudeEvent.
  </action>
