**src-tauri/src/process/signals.rs** — signal helpers using nix crate:
- `pub fn kill_process_group(pgid: i32, signal: Signal) -> Result<(), String>` — wraps `nix::sys::signal::killpg`
- `pub fn graceful_kill(pgid: i32) -> impl Future` — sends SIGTERM, sleeps 2s, sends SIGKILL
- `pub fn pause_group(pgid: i32) -> Result<(), String>` — SIGSTOP via killpg
- `pub fn resume_group(pgid: i32) -> Result<(), String>` — SIGCONT via killpg

**src-tauri/src/process/manager.rs** — core process operations:
- `pub async fn spawn(cmd: &str, args: &[String], cwd: &str, channel: Channel<OutputEvent>, state: tauri::State<'_, AppState>) -> Result<String, String>`:
  - Generate UUID task_id
  - Build `tokio::process::Command` with `pre_exec(|| { libc::setpgid(0, 0); Ok(()) })` for pgid isolation
  - Set `NO_COLOR=1`, `TERM=dumb` env vars
  - Pipe stdout and stderr
  - Spawn, get child pid (pgid = pid since setpgid(0,0))
  - Register ProcessEntry in state.processes
  - Spawn tokio tasks for stdout/stderr line reading, sending OutputEvent via channel
  - Spawn a waiter task: `child.wait()` then update ProcessEntry status and send OutputEvent::Exit
  - Return task_id

- `pub async fn cancel(task_id: &str, state: tauri::State<'_, AppState>) -> Result<(), String>`:
  - Look up pgid from state.processes
  - Call `graceful_kill(pgid).await`
  - Update status to Failed("Cancelled")

- `pub fn pause(task_id: &str, state: tauri::State<'_, AppState>) -> Result<(), String>`:
  - Look up pgid, call `pause_group(pgid)`
  - Update status to Paused

- `pub fn resume(task_id: &str, state: tauri::State<'_, AppState>) -> Result<(), String>`:
  - Look up pgid, call `resume_group(pgid)`
  - Update status to Running

**src-tauri/src/commands/process.rs** — four Tauri commands with specta:

#[tauri::command]
#[specta::specta]
pub async fn spawn_process(
    cmd: String,
    args: Vec<String>,
    cwd: String,
    on_event: Channel<OutputEvent>,
    state: tauri::State<'_, AppState>,
) -> Result<String, String> {
    process::manager::spawn(&cmd, &args, &cwd, on_event, state).await
}

#[tauri::command]
#[specta::specta]
pub async fn cancel_process(
    task_id: String,
    state: tauri::State<'_, AppState>,
) -> Result<(), String> {
    process::manager::cancel(&task_id, state).await
}

#[specta::specta]

pub fn resume_process(

}
