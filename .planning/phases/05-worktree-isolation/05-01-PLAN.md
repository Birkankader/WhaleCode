  <name>Task 2: Implement WorktreeManager with create, remove, list, and cleanup</name>
  <files>src-tauri/src/worktree/manager.rs, src-tauri/src/lib.rs</files>
  <behavior>
    - create_for_task(task_id) creates a worktree at `{project}/../.whalecode-worktrees/whalecode-{task_id_prefix}` with a branch `whalecode/task/{task_id_prefix}`
    - create_for_task returns WorktreeEntry with correct path and branch name
    - create_for_task fails gracefully if worktree path already exists (prunes stale, retries)
    - remove_worktree(worktree_name) prunes the worktree and deletes its branch
    - list_worktrees() returns only whalecode-prefixed worktrees
    - cleanup_stale_worktrees() finds and prunes invalid/prunable whalecode worktrees, returns list of cleaned names
    - Tests use temporary git repos (tempdir + git init) for deterministic behavior
  </behavior>
  <action>
    1. Create `src-tauri/src/worktree/manager.rs` with `WorktreeManager` struct:
       - Field: `repo_path: PathBuf`
       - `new(repo_path: PathBuf) -> Self`
       - `worktree_base_dir(&self) -> PathBuf` ‚Äî returns `{repo_path}/../.whalecode-worktrees`
       - `create_for_task(&self, task_id: &str) -> Result<WorktreeEntry, String>`:
         - Open repo via `Repository::open(&self.repo_path)`
         - Compute `worktree_name = format!("whalecode-{}", &task_id[..8])` (use min(8, len) for safety)
         - Compute `branch_name = format!("whalecode/task/{}", &task_id[..8])`
         - Compute `worktree_path = self.worktree_base_dir().join(&worktree_name)`
         - If worktree_path exists, attempt to prune stale worktree first, then remove dir
         - Create branch from HEAD commit
         - Create worktree with `WorktreeAddOptions::reference()`
         - Return `WorktreeEntry`
       - `remove_worktree(&self, worktree_name: &str) -> Result<(), String>`:
         - Find worktree, prune it
         - Remove the worktree directory if it still exists
         - Delete the associated branch (`refs/heads/whalecode/task/{id}`)
       - `list_worktrees(&self) -> Result<Vec<String>, String>`:
         - List all worktrees, filter to `whalecode-` prefix
       - `cleanup_stale_worktrees(&self) -> Result<Vec<String>, String>`:
         - Iterate worktrees with `whalecode-` prefix
         - Check `validate()` and `is_prunable()`
         - Prune invalid/prunable ones
         - Return names of cleaned worktrees

    2. Add `mod worktree;` to `src-tauri/src/lib.rs` (after `mod context;`)

       - Test: worktree directory exists after creation
       - Test: remove_worktree cleans up directory and branch
       - Test: list_worktrees returns only whalecode-prefixed entries
       - Test: cleanup_stale_worktrees handles invalid worktrees
       - Test: create_for_task with duplicate task_id recovers gracefully
  </action>
